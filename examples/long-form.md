- "저희가 원하는 수준의 시니어 엔지니어를 찾는 것이 생각보다 훨씬 더 까다로운 일이었습니다" (단락 7) — "훨씬 더" 강조어가 두 개 겹쳤지만 회고에서 강조해야 할 부분. 단락 7 변경 0 곳.
- 모든 숫자·고유명사·인용문(투자자 발언 등)은 글자 단위로 그대로 보존.
- "감사합니다" 마무리는 회고 도메인에서 의도된 격식 — 판에 박힌 마무리 cat 5 에서도 "상황에 맞을 때만" 룰에 따라 살림.

## 적용 가이드

이 사례가 보여주는 것:

- **장문일수록 룰은 덜 빡빡하다.** 단락당 3곳·전체 20 % cap 안에서도 글이 길면 자연스레 건드리는 비율이 떨어진다 — 17.3 % 정도가 회고 글의 자연스러운 humanize 강도다.
- **단락별 분포가 균등하지 않아도 된다.** 단락 6, 7 처럼 "AI 티가 안 나는 단락"은 0 곳도 OK. 강제로 모든 단락을 손대면 오히려 톤이 어색해진다.
- **숫자·고유명사·인용문은 글자 단위로 동일.** "30억 원 / 47곳 / 240만 원 / 18개월 / 12개 / 50명 / 38점 / NPS / MRR / GA / 시리즈 A" 모두 보존.
- **결론적으로 / 다양한 / 매우** 같은 패턴은 글 전체에서 1-2 회만 정리해도 충분히 인상이 달라진다 — 모든 출현을 다 잡지 않는다.
